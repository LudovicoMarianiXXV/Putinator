Pseudocodice Javascript principale:

                Funzione startGame():
                    Inizializza variabili:
                        askedQuestions = []
                        askedTypes = Set()
                        remainingCharacters = copia di characters
                        currentQuestions = copia di base_questions
                        alternativeQuestions = copia di currentQuestions
                        nextPhase = false
                    Ottieni una domanda casuale e mostralo
    
                Funzione getRandomQuestion():
                    Filtra le domande non ancora fatte e con tipo non già chiesto
                    Se ci sono domande rimanenti:
                        Scegli una domanda casuale
                    Altrimenti:
                        Se nextPhase è false:
                            Imposta nextPhase a true
                            Scegli domande dalla lista questions
                            Riprova a selezionare una domanda
                    Ritorna null se non ci sono più domande
    
                Funzione displayQuestion(question):
                    Mostra la domanda sulla pagina
                    Assegna le azioni per ogni risposta (yes, no, dontKnow)
    
                Funzione handleAnswer(answer, question):
                    Aggiorna l'immagine in base alla risposta
                    Aggiungi la domanda alla lista delle domande fatte
                    Aggiungi il tipo alla lista dei tipi chiesti
                    Filtra la lista dei personaggi rimanenti
                    Se c'è un solo personaggi rimanente, mostra il risultato
                    Altrimenti, seleziona e mostra la prossima domanda
    
                Funzione updateCharacters(answer, question):
                    Filtra i personaggi in base alla risposta
                    Se la risposta è "dontKnow", rimuovi la domanda dalla lista delle domande fatte
    
                Funzione displayResult():
                    Se c'è solo un personaggio rimanente, mostra il risultato "vinto"
                    Se ci sono più personaggi, mostra il risultato "perso"
                    Salva il risultato nel localStorage
                    Reindirizza alla pagina del risultato
    
                Evento DOMContentLoaded:
                    Inizia il gioco chiamando startGame()

Pseudocodice per la gestione del risultato:

                Leggi il risultato dal localStorage usando la chiave "gameResult"
                Convertilo in un oggetto JavaScript (resultData)
    
                Se resultData esiste:
                    Se resultData.guessed è vero (gioco vinto):
                        Se esiste resultData.character.image:
                            Imposta l'immagine a sinistra usando resultData.character.image
                            Imposta l'alt text dell'immagine con resultData.character.name
                        Crea un elemento h2 con il nome del personaggio (resultData.character.name)
                        Aggiungi l'elemento h2 al contenitore (container)
                    Altrimenti (gioco perso):
                        Crea un elemento h2 con il messaggio resultData.message
                        Aggiungi l'elemento h2 al contenitore (container)
                Altrimenti (se non ci sono risultati):
                    Imposta il contenuto del contenitore a "Nessun risultato disponibile."
